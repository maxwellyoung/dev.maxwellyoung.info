# Career OS (3-Year 10x Plan)

This folder turns your master plan into a weekly operating system.

## What this implements

Core artifacts:
- `01-positioning-statement-v1.md`: one-line niche + problem + outcome + audience
- `02-opportunity-scorecard.md`: scoring model for roles/clients/projects
- `03-proof-portfolio.md`: case-study tracker (6-12 proof assets)
- `04-weekly-scoreboard.md`: lead + lag KPI tracking
- `05-quarterly-experiment-brief.md`: hypothesis and stop/continue logic
- `06-offer-sheet.md`: premium offer definition and price floor

Execution layer:
- `07-14-day-launch.md`: day-by-day launch sequence
- `08-90-day-review.md`: acceptance tests and reset protocol
- `09-contact-pipeline.md`: 30-target relationship pipeline
- `10-authority-artifact-pipeline.md`: 12-week proof/content pipeline
- `11-income-model.md`: baseline-to-10x revenue model and checkpoints

## Setup (60 minutes)

1. Complete `01-positioning-statement-v1.md`.
2. Populate `02-opportunity-scorecard.md` scoring weights and thresholds.
3. Add your baseline in `04-weekly-scoreboard.md` and `11-income-model.md`.
4. Draft your first offer in `06-offer-sheet.md`.
5. Start Day 1 in `07-14-day-launch.md`.

## Weekly cadence (8-12h)

- `4h` Deep skill build
- `2h` Proof production
- `2h` Pipeline development
- `2h` Asset build
- `1-2h` Weekly review and plan

## Rules

- Ship one needle-moving deliverable each week.
- Run at most two active experiments at once.
- Maintain a stop-doing list and update it weekly.
- Protect core job performance as downside control.

## Decision defaults baked into this system

- Objective: income growth
- Horizon: 3 years
- Path: hybrid (job + owned assets)
- Risk: moderate
- Capacity: 8-12h outside core job
- Moat focus: technical specialization
- Priority order: salary leverage -> side revenue -> scalable asset
