# Opportunity Scorecard

Score every new opportunity before saying yes. Minimum threshold: 60/100.

## Scoring weights

| Criteria | Weight | Description |
|---|---:|---|
| Rate / revenue | 25 | Does it pay enough? ($50+/hr or meaningful equity) |
| Craft quality | 20 | Will I ship work I'm proud of? (interaction design, not CRUD) |
| Learning | 15 | New tech, new domain, new skill? |
| Portfolio value | 15 | Can I show this publicly? Will it impress? |
| Time fit | 15 | Compatible with uni + Silk + music schedule? |
| Relationship | 10 | Do I like these people? Future referrals? |

## Threshold

- **70+**: Strong yes
- **60-69**: Proceed with caution, negotiate terms
- **Below 60**: Decline unless strategic (relationship play, door-opener)
