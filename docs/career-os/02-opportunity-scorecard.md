# Opportunity Scorecard

Score every role/client/project before saying yes.

## Scoring model

Rate each dimension `1-5`. Multiply by weight. Sum to total out of 100.

| Dimension | Weight | Score (1-5) | Weighted |
|---|---:|---:|---:|
| Income upside (12-month) | 30 |  |  |
| Skill moat compounding | 20 |  |  |
| Proof visibility (portfolio/public-safe) | 15 |  |  |
| Network leverage (access to decision makers) | 15 |  |  |
| Time efficiency (ROI per hour) | 10 |  |  |
| Burnout risk (reverse score) | 10 |  |  |
| **Total** | **100** |  |  |

## Decision thresholds

- `80-100`: aggressively pursue
- `65-79`: pursue if capacity exists
- `50-64`: hold/waitlist
- `<50`: reject

## Red-line filters (auto reject)

- Requires >12h/week outside core role for >4 weeks.
- No measurable outcomes available.
- No reusable asset/proof generated.
- Damages core job performance.
